## 🚀 Sobre nós

A **STD Out** é uma fábrica de software focada em criar soluções digitais robustas, escaláveis e orientadas a resultado.

Nosso compromisso é transformar ideias em sistemas reais, utilizando arquitetura moderna, boas práticas e foco total em valor de negócio.

> Excelência não é um objetivo. É o padrão.

---

## 🧠 O que fazemos

- 🏗️ Desenvolvimento de sistemas sob medida  
- 🌐 Aplicações web modernas (SPA, SSR, APIs)  
- 🤖 Soluções com Inteligência Artificial  
- ⚙️ Arquitetura e DevOps  
- 📊 Plataformas de dados e analytics  
- 🔐 Segurança e boas práticas  

---

## 🧰 Stack principal

```bash
Frontend:   Vue.js, Nuxt, Tailwind, Vuetify
Backend:    Node.js, PHP (Laravel/Lumen)
Database:   PostgreSQL, MySQL
Infra:      Docker, Traefik, Apache, Linux
AI/Data:    LLMs locais, Qdrant, pipelines de dados
```
